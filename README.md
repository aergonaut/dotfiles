# `dotfiles`

[My](https://dev.to/aergonaut) dotfiles. I use this repository to synch dotfiles
between my work and home machines, and to make upgrading machines easier.

These files are managed by the [`rcm`](https://github.com/thoughtbot/rcm) tool,
which makes symlinking dotfiles between a repository and my home directory easy.

## Getting Started

1.  Install Homebrew

2.  Install `rcm`

    ```
    brew tap homebrew/formulae
    brew install rcm
    ```

3.  Clone the repository into `.dotfiles`

    ```
    git clone https://github.com/aergonaut/dotfiles.git .dotfiles
    ```

4.  Run `rcup`
